n Bietjie agtergrond oor HTML

HTML - Hypertext Markup Language - is n programmeering taal wat gebruik word as die fondasie van die internet, alle webwerve reg om die werelg gebruik HTML vir hulle webwerf. Daar is nog programmeering tale wat gebruik word vir websites naamlik: python, javascript, css, ens. Op hierdie webwerf gaan jy leer hoe om van HTML gebruik te maak.


Wat gaan u leer?

By hierdie werbwerf sal u leer hoe om n webwerf sal skryf, spesifiek hierdie webwerf!


Voordat ons begin

Maak n txt file op jou rekenaar op, dan klik op file en klik save as en dan stoor dit as naam.html, vervang naam met die naam van die file. Klik dan een keer op die html file en reg klik dan op dit, klik open as en klik dan op notepad ( of enige ander text editor as jy weet wat jy doen soos notepad++ ), as jy die file wil stoor dan klik op ctrl+s.
Laat ons begin!


HTML kode


Elemente

<html> </html>

Eerstens, in html bestaan daar elements soos <html>, alle elements wat text bevat kort n sluiting wat gedoen word deur n / tussen die element se naam en die < te sit, bv: </html>. Binne die element begin en die sluiting van die element vind ons die "code block".

Vir enige html kode moet mens eers n <html> gevolg deur n </html> he, alles daar in is die html kode wat in ons webwerf gaan werk. van daar af kan mens nog elemente byvoeg, gewoonlik sit mens n element met die naam head en body in die eerste html element se code block. Jy sal nou sien wat daai elemente doen en hoe om hulle te gebruik.
